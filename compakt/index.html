FAQ

How much does it cost?

It's free!

Is it open-source?

Absolutely! Check out the repository here.

Do you send usage reports or track where I go online?

No. We constrain the extension to only work on Twitch. There's no code that monitors and tracks your usage. Please feel free to explore the source code yourself!

How does it work?

compakt uses suffix arrays to compress messages and minhashing to condense similar messages.

In the example GIF above, noticed the 2 similar messages "This is a message." and "This is an message.". compakt notices that the messages are similar enough and thus condensed them together.
